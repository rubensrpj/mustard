# /report-weekly - Weekly Report

## Trigger

`/report-weekly`

## Description

Generates a weekly progress report.

## Content

1. **Week Summary**
   - Total commits
   - Features completed
   - Bugs fixed

2. **Entity Changes**
   - New entities created
   - Entities modified

3. **Code Metrics**
   - Lines added/removed
   - Files changed

## Output Format

Markdown report suitable for team updates.
