# /status - Consolidated Status

## Trigger

`/status`

## Description

Shows consolidated project status.

## Information

1. **Git Status**
   - Current branch
   - Modified files
   - Commits pending push

2. **Pipeline**
   - Active pipeline (if any)
   - Current phase

3. **Build**
   - Last validation result

4. **Entity Registry**
   - Number of entities
   - Last update
