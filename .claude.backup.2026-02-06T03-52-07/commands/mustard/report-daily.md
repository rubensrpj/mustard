# /report-daily - Daily Report

## Trigger

`/report-daily`

## Description

Generates a daily progress report.

## Content

1. **Commits Today**
   - List of commits made
   - Files changed

2. **Pipeline Activity**
   - Features started/completed
   - Bugs fixed

3. **Pending Items**
   - Open pipelines
   - Uncommitted changes

## Output Format

Markdown report suitable for sharing or documentation.
